# AI Lost & Found System

A beautiful web application for managing lost and found items with image upload capabilities.

## Features

- User authentication with name and DOB
- Report lost items with photos
- Report found items with location and photos
- View all items in a beautiful card layout
- Delete items
- Animated background with falling items
- Glowing particle effects

## Local Setup

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Run the application:
```bash
python app.py
```

3. Open browser to `http://localhost:5000`

## Deployment

This app can be deployed to Render, PythonAnywhere, or Heroku.

## Technologies

- Flask (Python web framework)
- SQLite (Database)
- HTML/CSS/JavaScript
- Base64 image encoding
